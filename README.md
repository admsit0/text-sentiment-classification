# Clasificación de texto para análisis de sentimientos - Práctica 2

**Autores:** Adam Maltoni, Ibón de Mingo
**Asignatura:** Procesamiento del Lenguaje Natural (Grado en Ciencia e Ingeniería de Datos, UAM)

Pipeline de 5 ejercicios para la clasificación de sentimiento (Positivo/Neutro/Negativo) de reseñas de juegos de mesa de BGG. Incluye extracción de características, vectorización (TF-IDF y features lingüísticas), entrenamiento de modelos (SVM, RF, NB) y evaluación.
